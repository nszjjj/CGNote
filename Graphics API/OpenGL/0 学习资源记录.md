## CLion配置glfw

由于CLion自带CMAKE，在本地构建GLFW的时候（尤指学习LearnOpenGL的时候）是无需下载CMake的，直接用CLion打开Source Code构建即可，参考：

[CSDN - 使用CLion编译glfw并配置OPenGL或Vulkan环境](https://blog.csdn.net/weixin_44615552/article/details/130130919)
